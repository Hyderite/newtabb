How to install the extension

WARNING: CRX files might not work as they are not verified by Google. Also, NewTabb Extension does not work on mobile (unless you install Kiwi Browser. Press the three dots -> Extensions, and then follow the instructions below).

Step 1. Select the mirror:

Mirror 1 (Cloudflare, ZIP) Mirror 2 (Github, ZIP) Mirror 1 (Cloudflare, CRX) Mirror 2 (Github, CRX)
Click here to download the extension from Mirror 1 Click here to download the extension from Mirror 2 Click here to download the extension from Mirror 1 Click here to download the extension from Mirror 2

Step 2. Go to chrome://extensions (mobile users do not have to do this step)

Step 3. Enable "Developer mode"

Step 4. Unzip the file.

Step 5. Click on "Load unpacked".

Step 6. Select the unzipped folder.

You should be able to go to NewTabb when you open a new tab now.

Step 4. Drag the .crx file to the chrome://extensions tab.

You should be able to go to NewTabb when you open a new tab now.
